Introduction

This comprehensive guide covers speedrunning Resident Evil Remake (HD Remaster) as Jill Valentine in the Any% category on Normal difficulty.

The game that defined the survival-horror genre is back! Check out the remastered HD version of Resident Evil. In 1998, a special forces team is dispatched to investigate bizarre murders near Raccoon City. Attacked by a pack of vicious dogs, they take refuge in a haunted mansion—and quickly discover that survival is anything but guaranteed.

What is Any%

CATEGORY DEFINITION

Complete the game as quickly as possible using any legitimate means to reach the credits. This means you can use any glitches, skips, or strategies that don't involve external modifications to the game.

TIMING NOTICE

Times are tracked by the in-game timer (displayed on ending screen) rather than real time. For the purposes of racing, you may ignore IGT and use real-time instead.

Core Techniques Mastery

Before attempting the route, you must understand these fundamental techniques. These will save significant time throughout your run:

Enemy Baiting

Enemies (especially zombies) will "lunge" at the player when close enough. During this lunge there's a brief window to move away so the enemy misses, leaving them in recovery for roughly two seconds.

Stair Skating

Accomplished by mashing the run button whilst walking (d-pad forward) on stairs. Roughly 10Hz (10 button presses per second) is sufficient. This technique significantly speeds up stair traversal.

Pump Cancelling

After firing shouldered weapons like the shotgun and the grenade launcher, release aim/fire buttons whilst quickly pressing forward on d-pad immediately after firing.

Quick Shooting

When firing the handgun, there's a one-frame window where the gun can be fired again early up to three times. This requires precise timing but dramatically increases handgun DPS.

Splits and Timing Tools

Proper timing and split management is crucial for tracking your progress and identifying areas for improvement.

SETUP INSTRUCTIONS
  1. Install LiveSplit from livesplit.org
  2. Download the Jill Any% splits file from the Resources page
  3. Download the autosplitter script from the same location
  4. Open the splits file in LiveSplit
  5. Edit layout, add "Scriptable Auto Splitter" component
  6. Point it to the downloaded script file
BEFORE YOU BEGIN

This guide assumes you have completed Resident Evil Remake at least once and are familiar with the mansion layout, puzzle solutions, and basic enemy behaviors. If you're new to the game, consider doing a casual playthrough first.
